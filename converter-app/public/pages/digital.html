Decimal vs. Binary Units

Decimal Units (KB, MB, GB): Based on powers of 10 (1 KB = 1,000 bytes)
Binary Units (KiB, MiB, GiB): Based on powers of 2 (1 KiB = 1,024 bytes)

About Digital Units

Digital units are used to measure the size of digital information (data) in computing and telecommunications. The basic unit of digital information is a bit (binary digit), which can have a value of either 0 or 1. Eight bits make up one byte, which is commonly used as the fundamental unit of digital storage.

There are two main systems for measuring digital information:

  • Decimal System (SI Units): Uses powers of 10, with prefixes like kilo (10³), mega (10⁶), giga (10⁹), etc.
  • Binary System (IEC Units): Uses powers of 2, with prefixes like kibi (2¹⁰), mebi (2²⁰), gibi (2³⁰), etc.

The difference between these systems can lead to confusion. For example, a "gigabyte" might refer to 1,000,000,000 bytes (10⁹) or 1,073,741,824 bytes (2³⁰), depending on the context.

Unit Definitions

Decimal Units (SI)

  • Kilobyte (KB): 1,000 bytes
  • Megabyte (MB): 1,000 kilobytes = 1,000,000 bytes
  • Gigabyte (GB): 1,000 megabytes = 1,000,000,000 bytes
  • Terabyte (TB): 1,000 gigabytes = 1,000,000,000,000 bytes

Binary Units (IEC)

  • Kibibyte (KiB): 1,024 bytes
  • Mebibyte (MiB): 1,024 kibibytes = 1,048,576 bytes
  • Gibibyte (GiB): 1,024 mebibytes = 1,073,741,824 bytes
  • Tebibyte (TiB): 1,024 gibibytes = 1,099,511,627,776 bytes
